# Linguagem NPC com PLY
Projeto para disciplina de compiladores...

## Como rodar
1. Instale PLY: pip install ply
2. Rode lexer.py
3. Rode parser.py

## Exemplo de entrada
npc Goblin {
    vida = 20
    ataque = 5
}

## Explicação dos arquivos
- lexer.py : analisador léxico
- parser.py : analisador sintático
